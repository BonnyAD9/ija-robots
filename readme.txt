How to compile:
    mvn compile

How to run:
  mvn exec:java
    This will also print some other info.

How to build jar file:
  mvn package
    the jar file is target/ija-robots-1.0-SNAPSHOT.jar

How to build docs:
  mvn javadoc:javadoc
    docs are in folder target/site/apidocs
